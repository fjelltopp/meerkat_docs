meerkat_abacus.database_util.create_fake_data.create_form(fields, data=None, N=500, odk=True)[source]¶

Creates a csv file with data form the given fields

The types for fields are: {“integer”: [lower, upper]}

random int between upper and lower
{“one”: [“choice1”,choice2”,....]}
one random choice from the list
{“multiple: [“choice1”,choice2”,....]}
a random subset of choices
{“data: “key”}
a random choice from key in data

Args: from_name: name of the form fields: list of fields to include previous_data: data from other forms N: number of rows to generate odk: Does the form come from odk

Returns: list_of_records: list of dicts with data
